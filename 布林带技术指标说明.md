# 📊 布林带（BOLL）技术指标说明

## ✅ 已添加到主视图

### 什么是布林带？

布林带（Bollinger Bands，简称 BOLL）是由约翰·布林格（John Bollinger）在20世纪80年代发明的技术分析工具。它由三条线组成：

```
上轨（Upper Band） ━━━━━━━━  ← 中轨 + 2倍标准差
                  
中轨（Middle Band）━━━━━━━━  ← 移动平均线（MA）
                  
下轨（Lower Band） ━━━━━━━━  ← 中轨 - 2倍标准差
```

## 🎨 视觉配置

### 颜色方案

- 🟠 **上轨（Upper）**：`#FF6D00`（橙色）
- 🔵 **中轨（Middle）**：`#2196F3`（蓝色）
- 🟢 **下轨（Lower）**：`#00C853`（绿色）

### 参数设置

```javascript
{
  name: 'BOLL',
  calcParams: [20, 2],  // [周期, 标准差倍数]
  precision: 2          // 显示精度（小数点后2位）
}
```

**参数说明**：
- **周期（Period）**: 20 - 计算移动平均线的K线数量
- **标准差倍数（StdDev）**: 2 - 标准差的倍数，用于计算上下轨

## 📈 布林带计算公式

### 中轨（MB - Middle Band）
```
MB = SMA(Close, 20)
```
简单移动平均线（20周期）

### 上轨（UP - Upper Band）
```
UP = MB + 2 × STD(Close, 20)
```
中轨 + 2倍标准差

### 下轨（DN - Down Band）
```
DN = MB - 2 × STD(Close, 20)
```
中轨 - 2倍标准差

其中：
- `SMA` = 简单移动平均线
- `STD` = 标准差
- `Close` = 收盘价

## 🎯 技术分析要点

### 1. 布林带宽度

**收窄（Squeeze）**：
```
上轨 ┈┈┈┈┈┈┈
中轨 ━━━━━━━
下轨 ┈┈┈┈┈┈┈
```
- 表示市场波动性降低
- 通常预示着即将突破
- 可能是大行情的前兆

**扩张（Expansion）**：
```
上轨 ━━━━━━━━━━
中轨 ━━━━━━━
下轨 ━━━━━━━━━━
```
- 表示市场波动性增加
- 价格趋势强劲
- 行情正在进行中

### 2. 价格位置

**触及上轨**：
```
价格 ●
上轨 ━━━━━━━  ← 可能超买
中轨 ━━━━━━━
下轨 ━━━━━━━
```
- 可能处于超买状态
- 考虑获利了结
- 但强势行情可能沿上轨运行

**触及下轨**：
```
上轨 ━━━━━━━
中轨 ━━━━━━━
下轨 ━━━━━━━  ← 可能超卖
价格 ●
```
- 可能处于超卖状态
- 考虑买入机会
- 但弱势行情可能沿下轨运行

**在中轨附近**：
```
上轨 ━━━━━━━
中轨 ━━━●━━━  ← 震荡区域
下轨 ━━━━━━━
```
- 价格处于均衡状态
- 方向不明确
- 等待突破信号

### 3. 突破信号

**向上突破上轨**：
```
        价格 ↗ ●
上轨 ━━━━━━━
中轨 ━━━━━━━
下轨 ━━━━━━━
```
- 强势买入信号
- 趋势可能持续
- 注意回调风险

**向下突破下轨**：
```
上轨 ━━━━━━━
中轨 ━━━━━━━
下轨 ━━━━━━━
        价格 ↘ ●
```
- 强势卖出信号
- 下跌趋势可能持续
- 注意反弹风险

## 💡 交易策略

### 策略1：均值回归

**适用场景**：震荡市场

**操作方法**：
1. 价格触及上轨 → 考虑卖出
2. 价格触及下轨 → 考虑买入
3. 目标价格：中轨（MA20）

**风险**：在趋势市场中容易失效

### 策略2：趋势跟踪

**适用场景**：趋势市场

**操作方法**：
1. 价格突破上轨，沿上轨运行 → 持有多单
2. 价格突破下轨，沿下轨运行 → 持有空单
3. 价格回到中轨 → 止盈或减仓

**风险**：可能在震荡市场中频繁交易

### 策略3：布林带收窄

**适用场景**：寻找突破机会

**操作方法**：
1. 观察布林带宽度
2. 当带宽收窄至历史低位 → 准备
3. 等待价格突破上轨或下轨 → 跟随方向交易

**风险**：假突破可能导致损失

### 常用参数组合

**1. 标准布林带**
```javascript
calcParams: [20, 2]  // 默认配置
```
- 最常用的设置
- 适用于日线图
- 平衡灵敏度和可靠性

**2. 敏感布林带**
```javascript
calcParams: [10, 1.5]
```
- 反应更快
- 适用于短线交易
- 信号更频繁但可能更多噪音

**3. 稳健布林带**
```javascript
calcParams: [50, 2.5]
```
- 反应较慢
- 适用于中长线交易
- 信号更可靠但较少

## 📊 与其他指标配合使用

### BOLL + RSI

**组合逻辑**：
- BOLL 判断价格位置
- RSI 判断超买超卖

**买入信号**：
```
✅ 价格触及布林带下轨
✅ RSI < 30（超卖）
→ 强烈买入信号
```

**卖出信号**：
```
✅ 价格触及布林带上轨
✅ RSI > 70（超买）
→ 强烈卖出信号
```

### BOLL + MACD

**组合逻辑**：
- BOLL 判断价格位置
- MACD 判断趋势方向

**买入信号**：
```
✅ 价格在布林带下轨附近
✅ MACD 金叉
→ 趋势反转买入
```

**卖出信号**：
```
✅ 价格在布林带上轨附近
✅ MACD 死叉
→ 趋势反转卖出
```

### BOLL + 成交量

**组合逻辑**：
- BOLL 判断价格位置
- 成交量确认突破真实性

**有效突破**：
```
✅ 价格突破上轨/下轨
✅ 成交量放大
→ 突破有效，跟随趋势
```

**假突破**：
```
❌ 价格突破上轨/下轨
❌ 成交量萎缩
→ 可能假突破，谨慎对待
```

## 🎓 实战技巧

### 技巧1：布林带走平

当布林带三条线几乎平行时：
```
上轨 ━━━━━━━━━
中轨 ━━━━━━━━━
下轨 ━━━━━━━━━
```
- 市场处于横盘震荡
- 等待方向选择
- 不建议大仓位操作

### 技巧2：双重确认

不要仅凭布林带交易，需要：
1. 布林带信号
2. 其他技术指标确认
3. 成交量配合
4. 价格形态支持

### 技巧3：动态调整

根据市场环境调整参数：
- 牛市：可以适当放宽参数（如 BOLL(20, 2.5)）
- 熊市：可以适当收紧参数（如 BOLL(20, 1.5)）
- 震荡市：使用标准参数（BOLL(20, 2)）

## ⚠️ 注意事项

### 1. 不是万能指标

- ❌ 不要单独依赖布林带
- ✅ 应与其他指标配合使用
- ✅ 结合基本面分析

### 2. 滞后性

- 布林带基于历史数据计算
- 有一定的滞后性
- 快速变化的市场可能不适用

### 3. 假突破

- 价格可能突破后快速回落
- 需要等待确认
- 设置止损很重要

### 4. 不同周期

- 日线、小时线、分钟线的布林带会给出不同信号
- 建议使用多周期分析
- 以大周期为主，小周期为辅
